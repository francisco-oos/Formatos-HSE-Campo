# Ajustes finales beta2

Cambios aplicados sin modificar la lógica principal de generación de PDF, JSON, fotos ni QR.

## Tarjeta de Observación

- En **Tipo de observación** se agregó la opción `Otro`.
- Al seleccionar `Otro`, aparece un campo para especificar el motivo.
- En **Acción asignada a** ahora se usa el catálogo de áreas/departamentos:
  - Adquisición (Registro)
  - Perforación
  - Topografía
  - Gestoría
  - Logística
  - Jefatura
  - Inmuebles
  - QC

## Puestos / Categorías

Se agregaron al catálogo general:

- Op. Drone
- Técnico en Mantenimiento
- Ingeniero Electrónico
- Jefe de Departamento
- Oficina

Cuando se selecciona **Jefe de Departamento**, aparece el campo **Departamento del jefe**.

## Inspección de Chaleco Salvavidas

- Se agregó el campo **ID empleado**.
- El ID empleado se guarda en JSON y se imprime en PDF.
